一、同分异构现象

1、平面偏振光  ( plane-polarized light )

二、偏振光，旋光仪和比旋光度

2、光学活性 (optical activity ):  

物质能使偏振光发生偏转的性质。具有光学活性的物质称作光学活性物质。或者说该物质具有旋光性。

3、旋光仪和比旋光度（specific rotation)

旋光度（observed rotation）: 旋光活性物质使偏振光振动平面旋转的角度,用“α ”表示 。

使偏振光振动平面向右旋转称右旋（dextrorotatory),用“ + ”表示；使偏振光振动平面向左旋转称为左旋

（levorotatory)， 用左“ - ”表示。

   比旋光度（specific rotation)：在一定温度和波长（通常为钠光灯，波长为589 nm)条件下,样品管长度为 1dm,样品浓度为 1g/ml 时测得的旋光度。是一物理常数.

D : 钠光源,波长为 589nm；T : 测定温度，单位为oC

a: 实测的旋光度；l :样品池的长度,单位为 dm；

c :为样品的浓度,单位为 g/ml。

实物和镜像不能重合的性质称为手性，具有手性的分子称手性分子。

1、对映异构和手性

sp3杂化的碳，如果连有4个不同基团，则为手性碳，也称手性中心、不对称碳、不对称中心。

相互对映而不能重合的两种化合物彼此就互称为对映异构体(Enantiomers)，这种现象称为对映异构现象(Enantiomerism)。

对映异构体的物理性质只有旋光度不同，且其旋光度的大小相同，方向不同。熔点、沸点、溶解度、比重等物理性质都相同。化学性质，在非手性环境下也是相同的，在手性环境下或者与手性化合物反应，则其化学性质不同。

2、分子的对称性与手性 

（1）对称中心（i）

（2）对称面（σ ）

凡是具有对称中心或对称面的分子，都能与其镜像重叠，为无手性分子，这种分子称为对称分子。

    20世纪90年代后，许多国家规定，凡结构中具有不对称因素的药物，即“手性药物”，必须拆分其相应的立体异构体，并分别研究其药理、毒理和药物代谢性质。对已上市的消旋体药物，要重新评价其光学异构体的性质。对新申报的药品，一开始就要合成其光学异构体。

在绝大多数情况下,凡是不具有对称中心或对称面的分子，与其镜象不能重叠，分子有手性（charity），称为手性分子，又称为不对称分子。

最常用的是费歇尔投影式（Fischer Projection）

碳链一般竖直放置。上下表示向里；左右表示向外
