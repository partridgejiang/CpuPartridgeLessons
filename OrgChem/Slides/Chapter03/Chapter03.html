一种判定基团优先级的递归伪码

function CompareGroupPriority(GroupA, GroupB: TGroup): Integer;

begin

  Result := 0; 

  if FirstAtom(GroupA) <> FirstAtom(GroupB) then 

    Result := FirstAtom(GroupA) - FirstAtom(GroupB)

  else begin 

    // Compare first group on first atom of GroupA/B 

    Result := CompareGroupPriority(FirstGroup(GroupA), 
　　FirstGroup(GroupB));

    if Result = 0 then   // Same first group 

    begin 

      // Compare second group on first atom of GroupA/B 

      Result := CompareGroupPriority(SecondGroup(GroupA), 
　　　SecondGroup(GroupB));

      if Result = 0 then  // Same second group 

        Result := CompareGroupPriority(ThirdGroup(GroupA), 
　　　　ThirdGroup(GroupB));

    end;

  end;

end;

概念小结

 
  • 手性： 
    　    实物与其镜像不能重合的性质 
  •  
  • 手性分子 
  •  
  • 手性分子-对映异构(旋光异构)
 
  • 手性碳原子：连接四个不同基团的碳原子，最常见的手性中心 
  •  
  • 手性中心≠手性分子 
    有手性中心不一定是手性分子（如内消旋体） 
    手性分子不一定有手性中心（如轴手性、面手性化合物） 
    但绝大多数手性分子都具有手性中心，绝大多数有手性中心的分子也都有手性
 
 
  • 含一个手性中心的分子 
    •  
    • 具有手性，有一对对映异构体(+)(-)
    •  
    • 一对对映异构体等量混合 
      　　——      外消旋体(±)或dl
 
  • 含多个手性中心的分子 
  •  
    • 存在对映异构体、非对映异构体 
    •  
    • 可能手性中心旋光能力相互抵消 
      　　——      内消旋体(meso)
  •  
 
  •  
  • 环烷烃的顺反异构 
  •  
  • 顺反异构可能并发对映异构
 
  • 构象异构体 
    　　可以通过单键旋转互相转化。 
    　　这种转化的能垒很低，室温下可迅速进行。 
 
  • 构型异构体 
    　　一对构型异构体的转化必须通过化学键的断裂与重新组合才能完成。能垒很高，室温下不能进行。
